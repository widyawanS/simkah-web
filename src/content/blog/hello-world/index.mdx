## Masa Depan Web adalah SSR

Website ini menggunakan teknologi **Astro SSR** (Server-Side Rendering) yang dikombinasikan dengan **Keystatic** sebagai Content Management System. 

### Mengapa ini canggih?

1. **Kecepatan SSG, Fleksibilitas SSR:** Konten dibaca secara dinamis dari server saat ada permintaan, tapi tetap di-cache untuk performa maksimal.

2. **Kemandirian Total:** Anda tidak butuh database eksternal. Semua data disimpan sebagai file Markdown di server Anda sendiri.

3. **Desain Premium:** Menggunakan Bento Grid layout dan Glassmorphism untuk memberikan kesan website modern kelas atas.

Anda bisa merubah konten ini melalui dashboard `/keystatic`.
